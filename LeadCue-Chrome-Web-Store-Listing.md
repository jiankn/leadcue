## 3. Detailed Description

### Version A: 推荐正式版

LeadCue helps SEO, web design, and growth agencies turn company websites into qualified Prospect Cards.

Instead of giving you another contact database or random lead list, LeadCue analyzes the website you are already viewing and generates a clear Prospect Card with:

- Company summary
- ICP fit score
- Public contact points
- Website-based sales cues
- Outreach angles
- Copy-ready cold email first lines
- A short cold email draft
- Source-backed notes

Use LeadCue when you are researching potential clients and need a website-backed reason to reach out.

It is built for:

- SEO agencies
- Web design agencies
- Growth agencies
- Growth consultants
- B2B founders doing outbound

How it works:

1. Open a prospect website.
2. Click the LeadCue extension.
3. Get a Prospect Card with fit score, sales cues, and outreach angles.
4. Save the prospect, copy a first line, or export your research.

LeadCue is not a LinkedIn scraper, contact database, CRM, or email sending tool. It focuses on website-based prospect research before outreach.

Typical use cases:

- Find websites with weak CTAs or outdated design
- Spot SEO and content gaps
- Generate cold email first lines from real website observations
- Save qualified prospects for later outreach
- Export leads into your existing workflow

LeadCue helps you answer the question:

Why is this company worth contacting, and what should I say first?

## 8. Privacy Justification

Chrome review 可能关注网页读取权限。建议在隐私解释中写清楚：

```text
LeadCue only analyzes the active website when the user clicks the Analyze Website button. It does not run background scraping, collect browsing history, or sell user data.

Website content is used to generate prospect research, sales cues, and outreach suggestions for the user. Saved Prospect Cards are stored in the user's LeadCue account.
```

---

## 9. Permission Justification

### activeTab

```text
Used to read the active tab only after the user clicks Analyze Website, so LeadCue can extract website text, links, and metadata for prospect research.
```

### scripting

```text
Used to extract visible page content from the active tab after user action.
```

### storage

```text
Used to store lightweight extension preferences and login state.
```

### sidePanel

```text
Used to display LeadCue's prospect analysis next to the website being researched.
```

### host permissions

第一版尽量避免 broad host permissions。优先用 activeTab。

如果必须添加：

```text
Used only to analyze websites that the user explicitly chooses to scan.
```
